# Ampacity Renewables Solar Education

An interactive 3D educational platform for learning about utility-scale solar farm components and technology.

## Features

- **Interactive 3D Solar Farm** - Explore a realistic solar farm with monocrystalline panels, single-axis trackers, and full electrical infrastructure
- **6-Step Learning Path** - Guided tour through Module → String → Tracker → Combiner → Cabling → Inverter
- **Component Highlighting** - Click on any component to see it highlighted and learn about its function
- **Interactive Labels** - 19 clickable labels throughout the farm with detailed information
- **Sun Tracking Animation** - Watch the trackers follow the sun across the sky
- **Electricity Flow Visualization** - See animated particles showing DC (cyan) and AC (orange) power flow
- **10-Question Quiz** - Test your knowledge of solar farm technology
- **Responsive Design** - Works on desktop and mobile devices

- **HTML5/CSS3** - Structure and styling

##  Educational Content

Learn about:
- Photovoltaic (PV) solar panels and how they generate electricity
- Single-axis tracking systems (Nextracker & Array Technologies)
- String wiring and MC4 connectors
- Combiner boxes and their protection features
- DC and AC cabling systems
- String inverters and power conversion
- Site infrastructure (fencing, grounding, substations)
